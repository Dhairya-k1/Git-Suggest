# Git Suggest

**Git Suggest** analyzes your repository status and prints recommended Git commands directly to your VS Code terminal.

##  How to Use
- Type `gs` in the VS Code terminal of Git project.    
     
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp; OR  
  
- Open a Git project and press **`Ctrl`+`Alt`+`G`** (Mac: `Cmd`+`Alt`+`G`).


## Requirements
You must have Python and the `gitpython` library installed on your system:
```bash
pip install gitpython
```
